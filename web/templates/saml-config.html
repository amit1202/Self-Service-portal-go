Identity Provider (IdP) Settings

These settings should be provided by your Secret Double Octopus administrator.

The unique identifier for the IdP
The URL to send SAML authentication requests
The URL to send logout requests (optional)
The public certificate used to verify SAML responses

Service Provider (SP) Settings

These settings define how your application communicates with the IdP.

The unique identifier for your application
The URL where the IdP will send SAML responses
The URL where the IdP will send logout responses (optional)
The public certificate used to sign SAML requests
The private key used to sign SAML requests
